# DP-Jewels-Website

Premium digital showroom website for **D.P. Jewels** (Diamond Traders, Importers & Exporters).

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Tech Stack

- React + Vite
- Tailwind CSS (v4 via `@tailwindcss/vite`)
- Framer Motion
- React Router
- lucide-react

